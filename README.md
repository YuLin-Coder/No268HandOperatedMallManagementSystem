## 项目介绍
基于ssm+jsp的手办商城管理系统：前端 jsp、jquery、bootstrap，后端 maven、springmvc、spring、mybatis；角色分为管理员、用户；集成商品信息、购物车、我的订单、客服、商品管理等功能于一体的系统。
